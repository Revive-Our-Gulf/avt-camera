Getting Started

How to setup

To integrate the camera feed into the BlueOS Cockpit extension:

  1. In Cockpit go to 'Edit Interface'.
  2. Select the 'Regular' widget type in the widgets menu.
  3. Add a new iframe widget.
  4. Set the iframe source URL to: http://camera.local/cockpit.
  5. Adjust the dimensions as needed for optimal display.
