AVT Camera System

Welcome to the Allied Vision Technologies (AVT) camera control system.

About

This application provides a web interface to control AVT cameras, manage recording sessions, and access camera settings.

Key Features

  • Live camera preview
  • Record video in WRITE mode
  • Preview mode for setup
  • Adjust camera parameters
  • Access recorded files

Getting Started

Basic Operation
  1. Go to Cockpit to access the main camera interface
  2. Click to enter preview mode
  3. Enter a folder name where recordings will be saved
  4. Click to start recording
