About

This application provides a web interface to control the AVT camera, manage recordings, and access camera settings.

Getting Started

BlueOS Cockpit Integration

To integrate the AVT camera feed into the BlueOS cockpit, follow these steps:

  1. Navigate to the BlueOS cockpit widgets
  2. Add a new iframe widget.
  3. Set the iframe source URL to: http://camera.local/cockpit.
  4. Adjust the dimensions as needed for optimal display.
Basic Operation

The AVT camera operates in three modes: Standby, Preview, and Recording.

  • Standby Mode: The last captured frame is displayed, and the camera is idle.
  • Preview Mode: The camera streams the down camera using the configured stream resolution, but no data is saved.
  • Recording Mode: The camera saves full-resolution JPEG images in a folder named {date}_{time}_{transect_name}.
  1. Click to enter preview mode.
  2. Enter the transect name into the entry field where recordings will be saved.
  3. Click to start recording.
  4. Click to stop recording. This will also stop the preview mode.
